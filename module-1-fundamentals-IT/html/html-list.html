HTML List Example

This is an example of an HTML list:

  • Item 1
  • Item 2
  • Item 3

Here is an ordered list:

  1. First Item
  2. Second Item
  3. Third Item

And here is a description list:

Term 1
Description for term 1.
Term 2
Description for term 2.
Term 3
Description for term 3.

Lists can be nested:

  • Outer Item 1
    • Inner Item 1
    • Inner Item 2
  • Outer Item 2
  • Outer Item 3
